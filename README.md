# My Chat 🚀

A room-based messenger where users can join chat rooms using IP addresses or custom room codes. Built with Node.js, Express, and Socket.io.

## Features

- 🔗 **IP-based Rooms** - Join rooms using any IP address or custom code
- ⚡ **Real-time Messaging** - Instant message delivery using WebSockets
- 👥 **Multi-user Support** - Multiple users can join the same room
- 🎨 **Modern UI** - Clean, dark-themed interface
- 📱 **Responsive** - Works on desktop and mobile devices
